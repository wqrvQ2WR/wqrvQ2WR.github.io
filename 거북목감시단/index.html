실시간 거북목 감지

Teachable Machine AI가 웹캠으로 사용자의 자세를 실시간 분석합니다. 머리가 앞으로 나가면 즉시 감지!

경고음 + 알림

거북목이 감지되면 경고음과 브라우저 알림으로 알려줍니다. 10초 쿨다운으로 과도한 알림 방지.

탭 전환해도 계속 감시

사이드 패널에 고정되어 다른 사이트를 보는 중에도 자세를 계속 체크합니다.

실시간 확률 게이지

거북목 확률을 퍼센트와 게이지바로 실시간 표시. 80% 이상이면 자세 교정 안내.
